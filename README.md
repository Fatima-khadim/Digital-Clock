# 🕒 Digital Clock

A simple Digital Clock built using HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Features
- Live current time
- Current date
- Dynamic greeting (Morning, Afternoon, Evening, Night)
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap 4

## 📸 Preview
Displays the current time and date with a greeting based on the time of day.
